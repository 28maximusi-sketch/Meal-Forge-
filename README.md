Meal Forge — Планировщик питания на 7 языках
Meal Forge — коллекция из семи независимых реализаций планировщика питания, каждая из которых работает на своём языке программирования. Проект создан для демонстрации кроссплатформенного подхода и предоставления удобного инструмента для планирования рациона, подсчёта калорий и управления рецептами.

✨ Уникальные возможности (общие для всех версий)
🥗 База продуктов и блюд – храните информацию о калорийности, БЖУ (белки, жиры, углеводы) и порциях.

📅 Планирование на день – составляйте меню из завтрака, обеда, ужина и перекусов.

🔢 Автоматический подсчёт суммарной калорийности и макронутриентов за день.

🧠 Интеллектуальная подсказка – автодополнение названий продуктов при вводе.

🎲 Генерация случайного меню на основе имеющихся блюд (с учётом калорийного лимита).

💾 Сохранение и загрузка данных в JSON-файл.

📤 Экспорт плана в CSV для анализа в Excel.

🌐 Интерфейсы:

Командная строка (CLI) с богатым набором команд.

Графический интерфейс (GUI) в Python (Tkinter), Java (Swing), C# (WinForms).

Веб-интерфейс в PHP.

Использование (общий синтаксис)
CLI (все версии):

bash
meal add-food "Овсянка" --calories 120 --protein 5 --fat 3 --carbs 20
meal add-meal "Завтрак" --food "Овсянка" --portion 1 --date 2024-01-15
meal list --date 2024-01-15
meal stats --date 2024-01-15
meal random --calories-limit 600
meal export --output plan.csv
Интерактивный режим (CLI):
Запустите без аргументов – откроется меню.
